# Email Validator EV

## Overview
Email Validator EV is a Python script that allows users to validate email addresses using various third-party email validation services. The script supports ZeroBounce, Abstract, and Proofy APIs for email validation.

## Version and Support
- Version: 1.0
- Support: Demonofinternet0@gmail.com
- YouTube: [DemonofInternet](https://www.youtube.com/DemonofInternet)

## Features
- Validate a single email address or perform bulk email validation.
- Configure API keys for ZeroBounce, Abstract, and Proofy.
- Reset API keys to default values.
- Logging of validation results.
- ASCII art representation of a cat for visual appeal.

## Prerequisites
- Python 3.x
- Required Python packages: `requests`

## Usage
1. Clone the repository or download the script.
2. Open a terminal or command prompt.
3. Navigate to the directory containing the script.
4. Run the script using the command: `python script_name.py`

## Menu Options
1. **Validate with ZeroBounce**: Perform email validation using the ZeroBounce API.
2. **Validate with Abstract**: Perform email validation using the Abstract API.
3. **Validate with Proofy**: Perform email validation using the Proofy API.
4. **Configure Settings**: Configure API keys for ZeroBounce, Abstract, and Proofy.
5. **Exit**: Exit the script.

## Configuration Settings
- Configure API keys for ZeroBounce, Abstract, and Proofy.
- Reset API keys to default values if needed.
